# HR-analysis
1.  Project title / Headline
🧑‍💼 HR Analytics: Employee Attrition Analytics Dashboard
Developed an interactive Power BI dashboard to analyze employee attrition trends and identify key factors affecting retention.

2.  Short description / purpose
•This project focuses on analyzing employee attrition using an interactive Power BI dashboard.
•It helps identify key factors such as job role, salary, and age influencing turnover.
•The insights support HR teams in making data-driven decisions.
•Overall, it aims to improve employee retention strategies.

3.  The dashboard was built using the following tools and technologies:
📊 Power BI Desktop - Main visualisation platform for report creation.
🔄 Power Query - Performed data cleaning, transformation, and preparation for accurate analysis.
📐 DAX - Created calculated measures and KPIs to analyze attrition trends and key metrics.
📂 File Format - .pbix for development and .png for dashboard previews.

4.  Data Source
•Format: Excel / CSV
•Records: 1,470 employees
•Features: Age, Department, Salary, Job Role, Attrition, Years at Company, Job satisfaction
•Type: Structured HR dataset

5.  Features / Highlights

i) Business Problem : Organizations face high employee attrition, leading to increased hiring costs, loss of talent, and reduced productivity. However, identifying the key reasons behind employee turnover is challenging without proper data analysis.

ii) Goal of the Dashboard : To analyze employee data and identify patterns and factors contributing to attrition. The dashboard aims to provide clear, data-driven insights to help HR teams improve employee retention strategies.

iii) Key Visuals :

Attrition Rate Overview showing total employees, attrition count, percentage, average salary, average years at company
Education field-wise and Job Role-wise Attrition to identify high-risk areas.
Attrition by Age Group, Salary Slab, and Years at Company to uncover trends.
Interactive filters and charts for deeper analysis based on different employee attributes.

iv) Business Impact and Insights : The dashboard helps HR teams quickly identify major factors driving attrition--> 
   Employees with less than 2 years' tenure 
   Employees with less than or equal to 5k salary
   Employees of age group 26-35 left more
